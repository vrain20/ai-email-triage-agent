# 🤖 AI Email Triage Agent

An intelligent email automation system that automatically 
classifies and organizes Gmail emails using AI.

## 🚀 What it does
- 📬 Watches Gmail inbox every minute
- 🤖 Uses Google Gemini AI to classify emails
- 🏷️ Automatically applies labels:
  - 🔴 Priority_Urgent
  - ✅ Priority_Action  
  - 📋 Priority_FYI
  - 🗞️ Priority_Newsletter
- ✍️ Creates AI draft replies for urgent emails

## 🛠️ Tech Stack
- **n8n** - Workflow automation
- **Google Gemini AI** - Email classification
- **Gmail API** - Email reading and labeling
- **Google Cloud Console** - OAuth2 setup
- **Railway** - Cloud deployment (24/7)

## ⚙️ How to Run
1. Install n8n: `npm install -g n8n`
2. Import `Email_Triage_Agent.json` into n8n
3. Add Gmail OAuth2 credentials
4. Add Google Gemini API key
5. Activate the workflow!

## 📸 Demo
The robot automatically:
1. Reads new emails
2. Sends to Gemini AI for classification
3. Applies correct label
4. Creates draft reply for urgent emails
